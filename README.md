# My Portfolio

## Description
Ce projet est un portfolio personnel conçu pour présenter mes compétences, mes projets et mon expérience. Il inclut des animations dynamiques, une navigation fluide et un design moderne.

## Fonctionnalités
- **Thème clair/sombre** : Permet de basculer entre les thèmes pour une meilleure expérience utilisateur.
- **Animations dynamiques** : Intégration de `animate.css` et `wow.js` pour des animations fluides.
- **Validation côté client** : Vérification des formulaires pour garantir des entrées valides.
- **Optimisation des images** : Chargement paresseux pour améliorer les performances.
- **Accessibilité** : Utilisation des rôles ARIA et des balises sémantiques.
- **SEO amélioré** : Métadonnées et balises optimisées pour les moteurs de recherche.

## Instructions d'installation
1. Clonez le dépôt :
   ```bash
   git clone <URL-du-dépôt>
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur.

## Dépendances
- **Navigateurs supportés** : Chrome, Firefox, Edge, Safari.
- **Bibliothèques externes** :
  - `animate.css` : Pour les animations CSS.
  - `wow.js` : Pour déclencher les animations au défilement.

## Problèmes connus
- **WOW.js** : Assurez-vous que la bibliothèque est correctement chargée avant d'initialiser.
- **Compatibilité mobile** : Testé sur les appareils modernes, mais des ajustements peuvent être nécessaires pour les anciens appareils.

## Instructions pour les développeurs
1. **Installation locale** :
   - Installez un serveur local comme `Five Server` ou `Live Server` pour tester les modifications.
2. **Ajout de nouvelles compétences** :
   - Ajoutez les icônes dans le dossier `img/skills/`.
   - Mettez à jour la section `skills` dans `index.html`.
3. **Ajout de nouveaux projets** :
   - Ajoutez les images dans `img/projects/`.
   - Mettez à jour la section `projects` dans `index.html`.

## Tests
- **Accessibilité** : Vérifié avec des outils comme Lighthouse.
- **Performances** : Optimisé pour un chargement rapide grâce au lazy loading des images.
- **SEO** : Métadonnées ajoutées pour améliorer le référencement.
